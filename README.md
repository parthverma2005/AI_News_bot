#  AI News Telegram Bot

An automated AI-powered Telegram news bot that fetches the latest AI news, summarizes it using Groq + Llama 3, and sends beautifully formatted news digests directly to Telegram.

---

##  Features

- 📰 Latest AI news updates
- 🤖 AI-powered summarization using Groq
- 📤 Automatic Telegram posting
- ⚡ Fast and lightweight
- 🔥 Clean formatted news digest
- 🌍 Covers:
  - OpenAI
  - Google Gemini
  - NVIDIA
  - Anthropic
  - Robotics
  - AI Startups
  - Machine Learning

---

##  Tech Stack

- Python
- Groq API
- Telegram Bot API
- NewsAPI / GNews
- python-dotenv
